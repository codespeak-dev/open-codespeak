## Dev environment

Install `uv`, then

```
uv python install 3.11
uv sync
```

### Activate venv

Option 1. `source .venv/bin/activate`

Option 2. Install `direnv` to automatically load environments when you change dirs

Follow https://direnv.net/docs/installation.html
